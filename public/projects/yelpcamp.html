YelpCamp - Full-Stack Web App

March - June 2025

Source Code

Project Overview

YelpCamp is a full-stack web application for users to discover and review campgrounds. The platform includes user authentication, authorization, and CRUD functionality for campgrounds, comments, and reviews. The project was built during a Udemy bootcamp as an introduction to full-stack development.

Tech Stack

Frontend

  • HTML5
  • CSS3 / Bootstrap
  • JavaScript (ES6+)

Backend

  • Node.js
  • Express.js
  • EJS (Templating)

Database

  • MongoDB (Atlas)

Testing

  • Jest
  • Google Lighthouse

Deployment

  • Render
  • Cloudflare
  • CircleCI

Development

  • Git & GitHub

System Architecture

The application follows an MVC (Model-View-Controller) monolithic architecture. MongoDB is used for flexible data storage, Express.js for routing and middleware, and EJS for templating.

Key Features

  • User authentication & authorization
  • CRUD functionality for campgrounds, comments, and reviews
  • Google Maps API integration
  • Responsive UI with Bootstrap

Production & Performance

  • Hosting: Deployed on Heroku
  • Database: MongoDB Atlas
  • Traffic: 100+ monthly users
  • SEO: Google PageRank #8 for "YelpCamp"
  • Performance: Optimized using Google Lighthouse

Lessons Learned

  • First full-stack web application
  • Understanding of Express.js & MongoDB
  • Deployment process using Render & Cloudflare
  • Continuous Integration with CircleCI
