Project Overview

This portfolio is designed to showcase my projects and technical expertise. The site was initially developed using HTML, Sass, and JavaScript, with later optimizations for performance and SEO.

Performance & Analytics

  • Google Lighthouse
  • Google Analytics

Architecture & Design

Initially, the site was built with an Express.js backend, but later refactored to a static site architecture using HTML and JavaScript, hosted on AWS S3. This resulted in significant performance improvements.

UX/UI Considerations

The design was optimized for responsiveness using a mobile-first approach. Custom SVG assets were used for icons, and lazy-loading techniques were implemented to improve performance.

Performance & Metrics

  • Hosting: Deployed AWS S3 + CloudFront
  • SEO: Google PageRank #1 for "Javier Ju Portfolio"
  • Performance: Lighthouse score improved from 12 → 88
  • Security: Mozilla Observatory grade improved from F → B-

Lessons Learned

  • Refactoring from Express.js backend to a static site improved performance significantly
  • SEO optimization with Open Graph & Google Lighthouse
  • GitHub & CI/CD pipeline setup for automated deployments
